# **App Name**: Kirana Stores

## Core Features:

- Fixed Navigation: Responsive top navigation bar with Home, About, Login, Signup, Admin, and User links.
- Theme Toggle: Theme toggle button (sun/moon icon) for switching between Light and Dark Mode, with smooth transitions.
- Animated Background: Live-action nature scene background animation.
- Login Card UI: Centered login card with glassmorphism effect, soft drop shadows, and rounded corners.
- Role-Based Login Form: Mobile number input field with dropdown for selecting Admin or User roles.
- OTP Login and Signup: "Login with OTP" button and link to "Sign up".
- Smart Hints: Use an AI tool to generate usage hints for Admin/User accounts based on user role and time of day. 

## Style Guidelines:

- Primary color: Soft green (#8FBC8F), representing nature and freshness.
- Background color: Light grey (#F0F8FF) in light mode and dark grey (#36454F) in dark mode, creating a natural, calming effect.
- Accent color: Earthy brown (#A0522D) for interactive elements, providing warmth and grounding.
- Body and headline font: 'PT Sans', a humanist sans-serif for modern and readable text.
- Simple, clean icons for navigation and interactive elements, representing natural elements like leaves, sun, and moon.
- Clean, centered layout with a focus on simplicity and ease of use, optimized for mobile responsiveness.
- Smooth transitions and subtle animations for hover effects, theme toggling, and loading states.
